# Workflow Bundles

Consolidated and granular workflow bundles that orchestrate the 15 core skills for targeted development and operational scenarios.

## Development Workflow Bundles

### 1. Modern Frontend App Bundle
**Description**: Complete setup for a modern React/Next.js frontend with Tailwind CSS.
- **Key Skills**: `@frontend-agent`, `@nextjs-best-practices`, `@react-best-practices`, `@react-ui-patterns`, `@tailwind-patterns`

### 2. Fullstack Application Delivery
**Description**: End-to-end fullstack development, from architectural planning to backend and frontend implementation.
- **Key Skills**: `@pm-agent`, `@senior-fullstack`, `@frontend-agent`, `@backend-agent`, `@typescript-expert`

### 3. API & Backend Services
**Description**: Scalable backend infrastructure development, emphasizing clean code, security, and proper implementation patterns. 
- **Key Skills**: `@backend-agent`, `@auth-implementation-patterns`, `@typescript-expert`

### 4. Quality Assurance & Testing
**Description**: Holistic testing pipeline covering unit, E2E, and code review standards.
- **Key Skills**: `@qa-agent`, `@testing-qa`, `@code-reviewer`

### 5. Product Management & Setup
**Description**: Initial application planning, requirement breakdown, and UX design flow.
- **Key Skills**: `@pm-agent`, `@ui-ux-designer`, `@senior-fullstack`

## Usage

Each workflow bundle typically follows standard phases:
1. **Planning**: Use `@pm-agent` to establish scope.
2. **Architecture**: Use `@senior-fullstack` and `@ui-ux-designer`.
3. **Execution**: Use `@frontend-agent`, `@backend-agent`, and related tech stack best practices.
4. **Verification**: Use `@qa-agent` and `@code-reviewer`.

### Example Usages in Chat:

``` text
Use the "Modern Frontend App Bundle" to build a new Next.js 14 dashboard.
```

``` text
Run the "Fullstack Application Delivery" workflow for our new Task Manager system.
```
